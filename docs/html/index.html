READ ME

This is PHP implementation of QR Code 2-D barcode generator. It is pure-php LGPL-licensed implementation based on C libqrencode by Kentaro Fukuchi.

At 2025-02-21 was cloned from https://sourceforge.net/projects/phpqrcode/ to Platon Technologies public GIT repositary.

Current repository is https://github.com/platon-net/phpQRcode
